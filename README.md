# Yunzog Dashboard

## Project Structure

```
src/main/java/com/yunzog/dashboard
    ├── App.java
    ├── MainController.java
    ├── db/
    ├── dao/
    └── model/

src/main/resources/com/yunzog/dashboard
    └── main.fxml
```

## Requirements

* JDK 25 or newer
* Maven

## Run the Application

From dashboard/:

```
mvn clean javafx:run
```
